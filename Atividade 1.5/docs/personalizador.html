Documentação do Pacote Personalizador

O pacote personalizador fornece funcionalidades para formatação de texto utilizando a biblioteca Rich.

Módulo: layout

Funções para manipulação de layouts com Rich.

Funções:

  • layout_simples(texto, isArquivo=False)
    Mostra o texto em um layout simples.
  • layout_complexo(texto, isArquivo=False)
    Mostra o texto em um layout mais detalhado.

Módulo: painel

Funções para exibição de painéis com Rich.

Funções:

  • painel_simples(texto, isArquivo=False)
    Mostra o texto em um painel simples.
  • painel_com_titulo(texto, isArquivo=False)
    Mostra o texto em um painel com título e bordas coloridas.

Módulo: progresso

Funções para exibição de barras de progresso.

Funções:

  • barra_progresso_simples(texto, isArquivo=False)
    Mostra uma barra de progresso simples.
  • progresso_com_mensagem(texto, isArquivo=False)
    Mostra uma barra de progresso com uma mensagem personalizada.

Módulo: estilo

Funções para estilização de texto com Rich.

Funções:

  • texto_em_negrito(texto, isArquivo=False)
    Mostra o texto em negrito.
  • texto_em_italico(texto, isArquivo=False)
    Mostra o texto em itálico.

Script: main.py

Interface de linha de comando para executar funções dos módulos.

Uso:

python main.py <texto> -m [layout|painel|progresso|estilo] -f <função> [-a]

Opções:

  • -a: Indica que o texto é um arquivo.
  • -m: Escolhe o módulo (layout, painel, progresso, estilo).
  • -f: Escolhe a função do módulo.
